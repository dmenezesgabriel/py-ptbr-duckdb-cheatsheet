## Criando o ambiente

```sh
#  Git Bash
# Change Directory
cd Documents
# Make Directory
mkdir my_data_analysis
# Change Directory
cd my_data_analysis
# Create Virtual Environment
python -m venv venv
# Open Visual Studio Code
code .
```

## Visual Studio Code

1. **CTRL + SHIFT + P**
2. Abrir preferencias do usuário :Preferences: Open User Settings (JSON)
3. ```json
   {
     "terminal.integrated.defaultProfile.windows": "Git Bash"
   }
   ```
4. Salvar: **CTRL + S**
